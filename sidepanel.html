How to Use

  1. Navigate to app.send.tg and log in
  2. Go to the Transactions page
  3. Click Export Raw CSV or Export Koinly CSV
  4. Wait for the extension to scroll through all transactions and fetch details
  5. The CSV file will download automatically when complete

Tip: Use the Stop button if you need to cancel the export process.
